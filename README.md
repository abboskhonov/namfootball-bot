# NamFootball Bot

A Telegram bot for football content.

## Setup

1. Create a bot with [@BotFather](https://t.me/BotFather) and get your token.
2. Copy `.env.example` to `.env` and add your token:

```sh
cp .env.example .env
# Edit .env with your BOT_TOKEN
```

3. Install dependencies:

```sh
bun install
```

4. Run the bot:

```sh
bun run dev
```

## Commands

- `/start` — Start the bot
- `/help` — Show help
- `/info` — Show bot info
